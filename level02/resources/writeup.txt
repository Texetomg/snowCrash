Для этого задания понадобиться виртуальная машина с Windows 10.

- Для начала устанавливаем Wireshark и 7zip на Windows 10.
- Нажимаем правой кнопкой мыши по образу SnowCrash.iso > 7zip > Open Archive, далее открываем папку "casper"
- Открываем с помощью 7zip файл filesystem.squashfs, далее идем в home > user > level02
- Открываем файл level-2.pcap с помощью Wireshark
- После просмотра всех пакетов, можно заметить строку 'Password: ' в пакете 43
- Нажимаем на этот пакет правой кнопкой мыши и выбираем Follow > TCP Stream
- В нем видим следующую строку 'Password: ft_wandr...NDRel.L0L'
- В шестнадцатеричной системе точка обозначает '7f', в ACII таблице это символ 'DEL'
- Таким образом, необходимо убрать такое количество символов перед точками, сколько и самих точек, получим пароль 'ft_waNDReL0L'

Далее выполняем следующие команды по очереди:
`su flag02`
